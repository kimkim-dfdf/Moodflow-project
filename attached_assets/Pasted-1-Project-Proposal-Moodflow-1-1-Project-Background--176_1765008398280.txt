1. Project Proposal – Moodflow
1.1 Project Background

요즘 학생들과 사회초년생들 사이에서 정신 건강 문제에 대한 관심이 빠르게 증가하고 있으며, 감정 분석 서비스 수요 또한 지속적으로 커지고 있다. 인스타그램이나 X와 같은 SNS에서는 우울하거나 힘들 때 들을 음악, 읽을 책, 혹은 무엇을 해야 할지 추천을 요청하는 게시글이 흔하게 보일 정도로, 많은 사람들이 자신의 감정 상태에 맞는 선택지를 직접 찾아 결정하는 데 어려움을 느끼고 있다.

2025년 기준 기존 서비스들은 감정 기록, 감정 분석, 간단한 통계 혹은 패턴 인식에 치중되는 경우가 많고, 사용자의 현재 감정에 기반해 즉각적으로 적합한 책·활동·할 일 등을 자동으로 추천해주는 웹 기반 서비스는 매우 드물다. 이러한 상황을 확인하며, 사용자가 매번 감정에 맞는 정보를 스스로 탐색해야 하는 비효율을 줄이고자 하는 필요성을 느끼게 되었고, 이를 계기로 Moodflow 시스템을 개발하게 되었다.

1.2 Project Aims & Objectives
1.2.1 Project Aim

이 프로젝트의 목표는 사용자의 감정을 분석하여, 해당 감정에 적합한 독서 추천, 활동 추천, 해야 할 일 제안 등을 제공하는 감정 기반 맞춤 추천 웹 서비스를 구축하는 것이다. 사용자가 감정에 압도되지 않고, 즉시 실행 가능한 선택지를 빠르게 확인할 수 있도록 돕는 것이 핵심 목적이다.

1.2.2 Project Objectives

사용자의 감정 상태를 입력 또는 분석하여 즉각적인 추천을 제공하는 시스템 구현

감정별 맞춤 콘텐츠(책, 도서, 글, 할 일 등)를 자동으로 분류하고 제시

번거로운 탐색 과정을 줄이기 위한 직관적인 사용자 인터페이스 제공

사용자들이 추천받은 활동을 기록하고 재확인할 수 있는 기능 제공

감정 데이터에 기반해 지속적인 사용자 경험 향상을 위한 피드백 구조 마련

1.3 Market Research

현재 시장에는 다양한 감정 기록 애플리케이션이 존재하며, Mood Tracker, Moodfit, Daylio 등의 앱이 널리 사용되고 있다. 그러나 다음과 같은 한계점이 발견된다:

서비스	감정 기록	통계/패턴 분석	음악 추천	책/도서 추천	할 일 추천	감정 기반 즉각 추천
Daylio	O	O	X	X	X	X
Moodfit	O	O	일부	X	X	X
기타 Mood Apps	O	O	X	X	X	X

현재 대부분의 서비스는 감정 추적과 기본 분석에 머물러 있으며, 감정별 실질적인 행동(독서·활동·할일)까지 자동으로 매핑해주는 서비스는 존재하지 않는다.

이러한 연구 결과는 Moodflow가 충족해야 할 명확한 시장의 빈틈을 보여준다.

1.4 Problem Statements & Solutions
1.4.1 Problem Statements

(1) 감정에 따른 실질적 추천 기능의 부재
대부분의 감정 앱은 단순 감정 기록 수준이며, 감정 상태에 맞춰 즉각적인 추천을 제공하지 못한다.

(2) 감정과 활동 간 연결 부족
사용자들은 감정이 우울하거나 불안할 때 실제 무엇을 해야 하는지 직접 검색하거나 SNS에서 물어보는 등 비효율적인 과정을 반복한다.

(3) 콘텐츠 추천의 단편성
일부 앱은 명상이나 호흡법 정도를 제공하지만, 다양한 형태의 콘텐츠(책·글·할 일 등)를 종합적으로 연결하지 못한다.

1.4.2 Suggested Solutions

Moodflow는 위의 문제들을 해결하기 위해 다음 기능을 제안한다:

(1) 감정 기반 즉각 추천 엔진 제공

사용자의 감정 상태를 분석

감정별 최적화된 책·도서·활동·할일을 즉시 제안

추천 콘텐츠는 지속적으로 업데이트 및 확장

(2) 감정–행동 매핑 데이터베이스 구축

감정 유형(우울, 불안, 무기력, 스트레스 등)에 맞는 행동 라이브러리 설계

심리 연구, 사용자 패턴 등을 반영한 매칭 알고리즘 적용

(3) 실행 가능한 활동 리스트 제공

단순한 분석이 아닌 즉시 행동으로 옮길 수 있는 제안 제공

예: “우울함 → 짧은 산책 추천 + 차분한 음악 + 감정 정리 에세이 추천”

(4) 사용자 맞춤형 기록 및 피드백 기능 제공

사용자가 선택한 콘텐츠 기록

감정 변화 패턴 분석 후 더 정교한 추천 제공
